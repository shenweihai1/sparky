Patents

Index analysis method and system and computer readable storage medium
TAN QI;GAO JUNXIU;SHEN WEIHAI; (NO: CN108460077A)
Abstract：The invention relates to an index analysis method and system and a computer readable storage medium. The method comprises the steps that an analysis topic corresponding to a to-be-analyzed index is acquired, wherein the analysis topic comprises at least one analysis strategy, each analysis strategy comprises at least one analysis mode, each analysis mode comprises configuration information, and each piece of configuration information comprises source data information and target variation parameters configured by a user; and for each analysis mode, according to the source data information and the target variation parameters configured by the user and included in the analysis mode, at least one value of the target variation parameters is acquired, and the values of the target variation parameters acquired according to the analysis topics are output. Through the index analysis method and system and the computer readable storage medium, a multilayer analysis architecture can be establishedaccording to the analysis topics, the to-be-analyzed index is automatically analyzed, and index analysis efficiency is improved.
